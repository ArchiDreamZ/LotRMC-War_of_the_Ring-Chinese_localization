# War of the Ring 汉化仓库

## 仓库介绍
该仓库共有9个文件夹，其中wotrmc是魔戒大战的本体，里面分为lang文件和speech对话文件。

其中lotr里包含一句 甘道夫的对话文件，剩下其他7个文件夹是该附属融合了7个其他模组的源代码。

这7个模组中大部分模组已经汉化，目前仅有“伊加墨的财产”需要汉化。
