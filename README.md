# War of the Ring 汉化仓库

## 仓库介绍
该附属融合了7个其他模组的源代码，因此该库中，加上魔戒传承和魔戒大战本身，总共有9个模组的语言文件。但是这些模组中有些已经有人汉化过了，因此并不是所有的都需要汉化。
